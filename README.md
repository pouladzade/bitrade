# Bitrade Matching Engine

Bitrade is a matching engine for spot trading. It is a high-performance, low-latency system that matches orders in real-time.

## Features

- [x] Order matching
- [x] Trade execution
- [x] Market depth
- [x] Order book
- [x] REST API
- [x] gRPC API
- [x] Historical data
- [x] Order history
- [x] Trade history
- [x] Order status
- [x] Order cancellation
- [x] multiple market support
